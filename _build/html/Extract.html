Extract¶

This is something I want to say that is not in the docstring.

class Idamous.plugins.extract.Extract.Extract(idamous)¶
get_architecture()¶

Returns the architecture the file was compiled for.

Example:
file = random.out returns “x86-64”
get_compiler()¶

Returns the name of the compiler used to compile the file if present.

( For Cole/David

Command: objdump -s –section .comment filename Output:

Contents of section .comment: 0000 4743433a 20285562 756e7475 20342e38 GCC: (Ubuntu 4.8 0010 2e342d32 7562756e 7475317e 31342e30 .4-2ubuntu1~14.0 0020 342e3329 20342e38 2e3400 4.3) 4.8.4.
Comment:
You may want to look into seeing if objdump will just display ASCII output instead of looking into parsing out the extra hex.

)

Example:
file = random.out returns “GCC: (Ubuntu 4.8.4-2ubuntu1~14.0.4.3) 4.8.4.”
get_filetype(meme=True)¶

Returns the type of file for the current file based on a sequence of bits at the start of the file. This is considered the mime type.

Example:
#!/bin/bash Returns: text/x-bash;
get_sections()¶

Returns a list of section headers for a file.

( For Cole/David

Command: objdump -h filename Output:

0 .interp 0000001c 0000000000400238 0000000000400238 00000238 2**0
CONTENTS, ALLOC, LOAD, READONLY, DATA
1 .note.ABI-tag 00000020 0000000000400254 0000000000400254 00000254 2**2
CONTENTS, ALLOC, LOAD, READONLY, DATA
2 .note.gnu.build-id 00000024 0000000000400274 0000000000400274 00000274 2**2
CONTENTS, ALLOC, LOAD, READONLY, DATA

)

Example:

file = random.out returns [

‘.interp’, ‘.note.ABI-tag’, ‘.note.gnu.build-id’, ‘.gnu.hash’, etc

]

get_version()¶

Returns the version number of the binary if present.

Example:
file = random.out returns “5.4.0”
